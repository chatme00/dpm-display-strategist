# Missing Information Follow-Up Questions

Ask at most 3-5 questions total. Choose only questions that materially affect the dating profile strategy.

## Priority Order

1. Relationship goal and timeline.
2. Target partner.
3. Dealbreakers.
4. Proof of advantages.
5. Display materials and privacy boundary.

## Question Bank

### Relationship Goal

- 你现在更想要的是认真恋爱、1-2年内结婚，还是先了解但不急着确定？真实想法是什么？
- 如果遇到合适的人，你希望关系大概以什么节奏推进？

### Target Partner

- 你最想吸引哪类人？请用 3-5 句话描述，不用写得标准。
- 过去真正吸引过你的人有什么共性？过去相处不顺的人又有什么共性？

### Dealbreakers

- 哪三类情况是一旦出现就不继续了解的？
- 关于城市、结婚、生育和家庭边界，哪些是你不能妥协的？

### Advantage Proof

- 你最想展示的 3 个优势分别有什么具体例子能证明？
- 熟悉你的人通常会夸你哪一点？有没有具体场景？

### Display Materials

- 你现在手上有哪些可用照片或生活素材？例如头像、全身照、工作照、做饭/运动/旅行/居家细节。
- 哪些信息你不希望公开展示，只适合私聊后再说？

## If User Does Not Answer

Proceed with the report and include a short note:

“由于你在 X 方面的信息还不完整，下面的展示面方案会先按当前资料做稳妥版。后续补充 X 后，可以进一步细化照片组合和筛选表达。”
