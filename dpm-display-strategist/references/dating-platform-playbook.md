# Dating Platform Display Playbook

Use this reference when converting DPM modeling into a dating platform profile.

## Sections

- Profile job
- Avatar and 6-9 photo set
- Basic profile fields, self-introduction, screening expressions, tags, scripts, and common risks

## Profile Job

A dating platform profile must do four things:

1. Make the right person stop.
2. Make the client legible quickly.
3. Prove the most important strengths.
4. Filter mismatches before long conversations.

## Avatar and First Photo

Use:

- Clear face, natural expression, clean light.
- Stable and approachable temperament.
- No heavy filters, sunglasses, over-cropped face, or group-photo ambiguity.

Avoid:

- Too distant, too performative, nightclub/wealth signaling, excessive beauty filters.
- Photos that create a persona the client cannot sustain offline.

## 6-9 Photo Set

Recommend a balanced set:

1. Clear face photo.
2. Natural half/full body photo.
3. Daily-life detail that proves order or warmth.
4. Work/professional/stable-life signal.
5. Long-term hobby or habit.
6. Social warmth or friend perspective.
7. Optional value signal: reading, cooking, family-aware life, outdoor, pet, home.
8. Optional contrast/memory point.
9. Optional photo reserved for private chat if public exposure is sensitive.

If privacy is high, use:

- Back/side-view lifestyle scenes.
- Hands/objects/home details.
- Bio clarity and private-chat proof instead of overexposure.

## Bio Structure

Use a 5-part self-introduction structure:

1. Who I am in concrete terms.
2. What my normal life looks like.
3. What my real strengths look like in daily details.
4. What kind of relationship and partner I hope for.
5. A light conversation hook.

Example skeleton:

“现在在[城市]做[工作/生活状态]，生活节奏比较[真实关键词]。  
平时喜欢[长期习惯/生活细节]，这部分也挺像我的性格：[优势的具体体现]。  
比起很快进入关系，我更看重[关系价值]。希望认识一个[目标伴侣关键词]、愿意认真经营关系的人。  
如果你也喜欢[低门槛话题]，可以从这个话题开始聊。”

## Basic Profile Display

Basic profile fields should be warm but efficient:

- Occupation: write concrete work status, not only title.
- City: include current city and settlement direction if relevant.
- Relationship goal: write clearly but lightly.
- Hobbies: write 2-4 lived scenes rather than a keyword pile.
- Partner expectation: mention 1-2 important expectations, not a long demand list.

Bad:

- “本人真诚善良，工作稳定，爱好广泛，希望找三观一致的人。”

Better:

- “在上海做产品，工作节奏还算稳定。休息日通常会做饭、运动，偶尔找个安静地方喝咖啡。希望认识一个沟通舒服、愿意认真经营关系的人。”

## Screening Expressions

Good screening is clear but not aggressive:

- “更希望认识认真了解、关系节奏稳定的人。”
- “异地可以聊，但希望有明确结束异地的计划。”
- “比较看重沟通和情绪稳定，不太适合长期失联或冷处理的相处方式。”

Avoid:

- Long lists of demands.
- Sarcasm toward past dates.
- “非诚勿扰” style hostility unless the platform context requires directness.

## Tags

Choose tags that prove positioning:

- Stable life: 规律生活, 长期主义, 情绪稳定, 认真关系.
- Warmth: 会照顾人, 喜欢做饭, 家庭感, 陪伴感.
- Growth: 自我驱动, 持续学习, 事业稳定.
- Lifestyle: 运动, 徒步, 读书, 家居, 旅行, 咖啡, 展览.

Use only tags supported by actual client data.

## Interaction Scripts

Opening message should reference the other person's profile and show a low-pressure path:

- “看到你也提到[共同点]，感觉这个点挺适合展开聊聊。你平时更喜欢[轻问题A]还是[轻问题B]？”
- “你的资料里我比较注意到[具体细节]，我也挺看重[相关价值]。可以问问你是怎么形成这个习惯的吗？”

Reply style:

- Mirror their energy, answer one real detail, ask one easy question.
- Avoid interrogation, résumé exchange, or early heavy conditions before rapport.

## Common Risks

- Too many demands in bio -> reduces warm approach.
- Too little boundary -> attracts mismatches.
- Claims without proof -> chat drops after curiosity.
- Over-polished persona -> offline inconsistency.
- Too much privacy -> no trust anchor.
