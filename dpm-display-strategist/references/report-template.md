# Client-Facing Profile Package Template

Use this exact 8-section structure. Keep the report warm, practical, and specific. Do not show backend scores. The output should feel like a complete dating platform profile package, not a strategy memo.

## Sections

- 8-section client-facing profile package
- Basic profile and self-introduction materials
- Direct copywriting requirements
- Tone rules

## 1. 你的展示面核心定位

Include:

- 一句话定位.
- 3-5 个气质关键词.
- 适合被理解成什么样的人.
- This profile should attract who and reduce what kind of mismatch.

## 2. 基础资料展示

Output how the client should fill or present basic dating-platform fields. Include:

- 昵称/称呼方向.
- 年龄、城市、职业、学历、婚恋目标等基础信息的表达建议.
- 标签建议, using only traits supported by the intake.
- 哪些基础信息适合公开写, 哪些适合私聊后补充.
- Avoid cold résumé style; make the basic profile legible and warm.

## 3. 自我介绍完整文案

This is the core profile copy section. Provide a complete self-introduction package:

- 一版完整自我介绍, integrating真实优势、生活状态、爱好、关系态度、对另一半期许.
- 3 个可替换开头.
- 3 个可替换结尾.
- 优势表达：turn abstract strengths into concrete human details.
- 爱好表达：write hobbies as lived scenes, not lists.
- 对另一半期许：write expectations warmly, with boundaries but no pressure.

The copy should sound like the client, not like a consultant. Use `copy-style-adaptation.md`.

## 4. 你最值得展示的真实优势

List Top 5. For each:

- 优势是什么.
- 为什么对目标伴侣重要.
- 哪些具体细节能证明它.
- 应该放在基础资料、自我介绍、照片、标签还是私聊里.

Avoid empty praise. If evidence is weak, write “这个优势可以用，但需要用具体细节补强”.

## 5. 你适合吸引的目标伴侣画像与筛选边界

Cover both attraction and filtering:

- 适合吸引的伴侣画像：性格特征、生活方式、关系期待、现实条件.
- 对方为什么会被你吸引.
- 不适合投入的人群.
- 哪些人容易造成无效沟通.
- 哪些边界适合公开写进资料.
- 哪些边界适合私聊后确认.

Use plain language. Do not write zodiac or MBTI as destiny.

Be direct but not harsh.

## 6. 相亲平台照片与版面设计

Include:

- 头像建议.
- 首图建议.
- 6-9 张照片组合建议.
- 资料区块排序：基础资料、自我介绍、兴趣爱好、期许与边界.
- Which photos support which copy points.

Photo set format:

1. 清晰头像.
2. 半身/全身自然状态.
3. 生活秩序或居家细节.
4. 工作/专业感或稳定感.
5. 兴趣/长期习惯.
6. 社交/朋友视角或温度感.
7. Optional: family/life planning signal.
8. Optional: contrast or memory point.
9. Optional: private-chat proof material, not necessarily public.

## 7. 可直接使用的参考文案

Provide:

- 稳妥版完整资料文案.
- 温暖真人版完整资料文案.
- 有记忆点版完整资料文案.
- 基础资料短句.
- 优势/爱好/期许的模块化句子.
- 适合放在资料里的筛选表达.
- 不建议写的内容.

Each version should be usable as-is, have a different tone, and not overpromise. Avoid stiff phrases like “本人真诚善良、希望找三观一致的人”.

## 8. 互动承接话术与真实性提醒

Provide:

- 别人主动私信时的回复风格.
- 你主动发起时的开场白.
- 3-5 个低压筛选问题.
- 适合推进见面的表达.
- 哪些内容不能夸大.
- 哪些优势需要用细节证明.
- 哪些表达可能吸引错配对象.
- 哪些隐私信息建议后置.
- 如果信息不足, explain the boundary briefly.

Screening questions should feel natural, not interrogative.

## Tone Rules

- Say “建议”, “更适合”, “可以这样表达”.
- Avoid “你条件不够”, “市场竞争力低”, “低价值”, “高价值”.
- Do not moralize the client’s preferences; translate them into boundaries and strategy.
- When expectations are narrow, say “会缩小范围，但能减少无效沟通”.
- The final copy should be conversational and specific, not polished into a generic品牌文案.
